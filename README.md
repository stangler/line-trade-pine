# line-trade-pine

TradingView用のライントレード戦略と、バックテスト結果を自動スクレイピングするツールのセットです。

## 概要

過去3日間の高値・安値・終値・VWAPをもとに水平ラインを自動描画し、ブレイクアウト・リバウンド・トレンドフォローの3つのエントリーモードで取引するPine Script戦略です。Pythonスクリプトにより、複数銘柄のバックテスト結果を自動収集できます。

## 特徴

- **過去3日間のライン自動描画**: 各日のO/H/L/C/VWAPを水平ラインで表示
- **3つのエントリーモード**: ブレイクアウト、リバウンド、トレンドフォロー
- **方向別・モード別の詳細レポート**: ロング/ショート、各エントリーモードごとの勝率をリアルタイム集計
- **時間フィルター**:  trading時間の制限と引けの強制決済
- **動的TP/SL**: 直近の抵抗線/支持線を利確目標に、ATRベースのストップロス
- **自動バックテストスクレイピング**: Playwrightを使用した複数銘柄の一括結果収集

## セットアップ

### 1. TradingViewチャートへの適用

1. TradingViewにログイン
2. チャート画面を開く（推奨: 5分足チャート）
3. Pine Editorを開く
4. `daytrade_3day_vwap_strategy.pine`の内容をコピーして貼り付け
5. 「Add to Chart」をクリック

### 2. パラメータ設定

チャート上で戦略の設定アイコンから以下のパラメータを調整可能：

**エントリー設定**
- エントリーモード: ブレイクアウト / リバウンド / トレンドフォロー
- ブレイク確認足数（1〜3足）
- リバウンド用タッチ幅（ATR倍率）
- VWAPによる方向フィルターの使用
- ロング/ショートの許可

**SL（ストップロス）設定**
- SLモード: ATR / 固定Tick
- SL ATR倍率（0.5〜5.0）
- SL固定Tick数
- ATR期間

**時間フィルター**
- trading時間の開始・終了時刻
- 引けでの強制決済の使用と時刻

**表示設定**
- 過去3日ラインの表示/非表示
- 各ラインの色（1日前:黄色、2日前:オレンジ、3日前:グレー）

**レポート設定**
- 詳細レポートテーブルの表示/非表示

### 3. バックテストスクレイピングツールのセットアップ

#### 必要なライブラリのインストール

```bash
pip install playwright
playwright install chromium
```

#### 対象銘柄の追加

`urls.txt`に対象銘柄の東証コードを1行1銘柄で記載：
```
186A
215A
268A
```

#### 実行方法

```bash
python tv_backtest_scraper_linetrade.py
```

初回実行時は以下を手動で実施：
1. TradingViewにログイン
2. ライントレード戦略を適用した5分足チャートを開く
3. Strategy Testerパネルを表示
4. データウィンドウを開く

準備ができたらEnterキーを押下。複数銘柄のバックテスト結果がCSVファイルとして保存されます。

## 出力ファイル

スクレイピング実行時、銘柄ごとに以下の形式のCSVファイルを出力：

```
{銘柄コード}_{YYYYMMDD}.csv
```

CSVには以下が含まれます：
- Strategy Testerの主要指標（純利益、プロフィットファクター、勝率、総取引数など）
- 方向別サマリー（ロング/ショートの取引数、勝数、勝率）
- モード×方向別サマリー（ブレイクアウト/リバウンド/トレンドフォロー × ロング/ショート）
- Data Windowの生データ

## 戦略の仕組み

### 水平ライン

過去3営業日について以下を自動計算し、水平ラインを描画：
- 始値（Open）
- 高値（High）
- 安値（Low）
- 終値（Close）
- VWAP（出来高加重平均価格）

### エントリーロジック

**ブレイクアウト**: 過去3日間のラインを上方/下方に抜けた時点でエントリー

**リバウンド**: 直近の支持線/抵抗線に触れて反発した時点でエントリー

**トレンドフォロー**: VWAPでの方向判定と、ラインのブレイクを組み合わせてエントリー

### 利確・損切り

- **利確（TP）**: 直近の抵抗線（ロング時）/支持線（ショート時）
- **ストップロス（SL）**: ATRベースまたは固定Tick数

### 時間制御

- 設定したtrading時間内のみエントリー許可
- 引け時刻に全ポジションを強制決済（オプション）

## 注意事項

- Pine ScriptはTradingViewの Pine Script v6 で動作
- スクレイピングツールはPlaywrightを使用。ブラウザの手動操作が必要
- バックテスト結果はCSVで保存されるため、日付ごとにファイルが増える
- TradingViewのUI変更によりスクレイピングが失敗する可能性あり
